Давайте теперь разберемся, как сделать красную строку, то есть отступ первой строки текста.

 Для этого предназначено свойство **text-indent**. Давайте с его помощью сделаем красную строку для абзацев:

    \<p>
    длинный текст...
    \</p>

    \<p>
    длинный текст...
    \</p>

    \<p>
    длинный текст...
    \</p>

    p {
    text-indent: 50px;

    text-align: justify;

    width: 400px;
    }
